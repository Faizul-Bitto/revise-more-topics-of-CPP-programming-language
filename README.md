# C++ Programming Language - Revision Topics

A comprehensive collection of C++ programming examples covering fundamental and intermediate topics. This repository contains practical code examples organized by topic, including input/output operations, control structures, classes and objects, dynamic memory management, string manipulation, and more.

## Topics Covered

- Basic I/O operations and formatting
- Variables, type casting, and data types
- Control flow (if-else, ternary operator, switch-case)
- Built-in functions (min, max, swap)
- String handling and char arrays
- Dynamic memory allocation
- Object-oriented programming (classes, objects, constructors)
- Array operations and sorting
- String manipulation and STL functions

Each topic includes example code with input/output files for practice and reference.
